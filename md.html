Analizar Ecuación

El Objetivo

A diferencia de una calculadora normal que solo trabaja con números (2 + 2 = 4), este programa utiliza Álgebra Simbólica.

Le damos una ecuación física de movimiento (MRUA), le decimos qué datos conocemos y le pedimos que "despeje" la variable que falta, tal como lo harías en papel con lápiz.

Fase 1

Importar el Cerebro Matemático

No usamos Python básico para esto. Traemos a SymPy, una librería científica capaz de entender que x + x = 2x.

Pasa el cursor sobre symbols, Eq y solve para ver sus superpoderes matemáticos.

Python

from sympy import symbols, Eq, solve
La Lógica

La Función resolver_mrua


def resolver_mrua(arg1_dicc, arg2_variable_a_buscar):
    
    # Definimos variables simbólicas
    x, x0, v, t, a = symbols('x x0 v t a')

    # Declaramos la ecuación: x = x0 + vt + 0.5at^2
    EcuacionPosicion = Eq(x0 + v * t + 0.5 * a * t**2 - x, 0)

    # Despeje algebraico
    solucion_simbolica = solve(EcuacionPosicion, arg2_variable_a_buscar)[0]

    # Sustitución numérica
    resultado = solucion_simbolica.subs(arg1_dicc)

    return resultado

Abstracción

Aquí definimos los símbolos. Para Python x no vale 5 ni 10, vale "x". Es una entidad abstracta lista para ser manipulada.

El Despeje

La magia ocurre en solve. No importa si le pides hallar el tiempo, la aceleración o la posición. SymPy reordenará la ecuación automáticamente por ti.

Deep Dive

De la Pizarra al Editor

Para que Python entienda la física, debemos "traducir" el idioma matemático a sintaxis de código. Mira cómo cambia cada parte de la ecuación:

Matemáticas

$$x = x_0 + v t + \frac{1}{2} a t^2$$
  • Multiplicación implícita ($vt$)
  • Potencias con superíndice ($t^2$)
  • Igualdad directa ($=$)

Python (SymPy)

Eq(x0 + v*t + 0.5*a*t**2 - x, 0)
  • Uso explícito de * para multiplicar
  • Operador ** para elevar potencia
  • Formato Eq(lado_izq - lado_der, 0)
¿Por qué " - x, 0"? En SymPy, las ecuaciones se expresan mejor como una resta igualada a cero. Matemáticamente, si $A = B$, entonces $A - B = 0$.
El código pasa la $x$ (posición final) restando al otro lado para cumplir con este formato.
Fase Final

La Prueba Numérica

1. Definir el Contexto

datos = {x0: 0, v: 10, a: 2, t: 5}

Creamos un diccionario que mapea los Símbolos a Números Reales. "La aceleración es 2", "El tiempo es 5".

2. Obtener Respuesta

valor = resolver_mrua(datos, x)

Llamamos a la función enviando los datos y diciendo: "Quiero que encuentres x (Posición Final)".

¡Y así funciona!
